## Technologies used:

### Frontend

- TypeScript
- React.js
- HTML
- Tailwind CSS
- Axios
- Vite
- Eslint
- PostCSS

### Backend

- Express
- JSON Web Token
- Cors
- Dotenv
- Mongoose
- Docker
- Nodemon
- BCryptJS
- Cookie-Parser
- Multer

## How to start

Install Docker. Here is an example how it can be done for Windows [link](https://docs.docker.com/desktop/install/windows-install/)

```bash
# start app in client/
npm start

# start local airbnb-clone-app API server in api/
docker compose up
npm start
```
